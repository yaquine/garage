🚗 API Gestion de Garage

Cette application expose une API REST permettant de gérer :

Les Garages

Les Véhicules

Les Accessoires des véhicules

Elle est développée avec Spring Boot et documentée avec Swagger / OpenAPI afin de faciliter les tests et l'exploration 
des endpoints.

📚 Documentation de l'API

La documentation interactive de l'API est disponible via Swagger UI :

http://localhost:8080/swagger-ui/index.html


📂 Fichiers de test

Pour faciliter les tests via Swagger, trois fichiers JSON sont fournis :

GarageApiJson → exemples de requêtes pour les APIs Garage

VehiculeApiJson → exemples de requêtes pour les APIs Véhicule

AccessoireApiJson → exemples de requêtes pour les APIs Accessoire

Ces fichiers contiennent des payloads prêts à utiliser dans Swagger.
